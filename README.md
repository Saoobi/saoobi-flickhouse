## Setup

```sh
git clone https://github.com/Saoobi/saoobi-flickhouse.git
cd saoobi-flickhouse/
npm install
cd client
npm install
cd ..
// Create your .env with PORT and API_KEY

npm run dev

// Project now running on localhost:3000


```
